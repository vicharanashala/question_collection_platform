## Rules

- When a sub-task is completed, change `- [ ]` to `- [x]` in the task file
- Update this index if task status changes (Pending → In Progress → Completed)
- Before starting a task, check both this index AND the sub-task checkboxes in the task file
- If a task or any of its sub-tasks is already marked completed (`- [x]`), do not redo it — reply that it is already completed
